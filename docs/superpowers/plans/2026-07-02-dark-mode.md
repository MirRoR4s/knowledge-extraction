# 深色模式 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add dark mode toggle to Docsify static site using built-in `dark.css` theme.

**Architecture:** Single-file modification to `index.html` — add theme-switching button, inline CSS for button styling, and inline JS for CDN link swapping + localStorage persistence.

**Tech Stack:** Vanilla JS, CSS, Docsify CDN themes.

## Global Constraints

- Only modify `index.html`
- No new files or dependencies
- Use Docsify CDN themes: `vue.css` (light) and `dark.css` (dark)
- Persist preference in `localStorage` under key `knowledge-theme`

---

### Task 1: Add dark mode toggle to index.html

**Files:**
- Modify: `index.html` (entire file, 27 lines)

- [ ] **Step 1: Read current index.html**

- [ ] **Step 2: Add `id` to theme link, add toggle button + CSS + JS**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CS 知识萃取</title>
  <link rel="stylesheet" href="//cdn.jsdelivr.net/npm/docsify@4/lib/themes/vue.css" id="theme-link">
  <style>
    #dark-mode-toggle {
      position: fixed;
      top: 10px;
      right: 10px;
      z-index: 1000;
      background: rgba(0,0,0,0.08);
      border: none;
      border-radius: 50%;
      width: 36px;
      height: 36px;
      cursor: pointer;
      font-size: 18px;
      display: flex;
      align-items: center;
      justify-content: center;
      transition: background 0.2s;
    }
    #dark-mode-toggle:hover {
      background: rgba(0,0,0,0.15);
    }
  </style>
</head>
<body>
  <button id="dark-mode-toggle" title="切换深色/浅色模式">🌙</button>
  <div id="app">Loading...</div>
  <script>
    window.$docsify = {
      name: 'CS 知识萃取',
      loadSidebar: true,
      subMaxLevel: 3,
      auto2top: true,
      search: {
        placeholder: '搜索',
        noData: '无结果',
        paths: 'auto'
      }
    }
  </script>
  <script src="//cdn.jsdelivr.net/npm/docsify@4/lib/docsify.min.js"></script>
  <script src="//cdn.jsdelivr.net/npm/docsify@4/lib/plugins/search.min.js"></script>
  <script>
    (function() {
      var link = document.getElementById('theme-link');
      var btn = document.getElementById('dark-mode-toggle');
      var STORAGE_KEY = 'knowledge-theme';

      function setTheme(theme) {
        var isDark = theme === 'dark';
        link.href = isDark
          ? '//cdn.jsdelivr.net/npm/docsify@4/lib/themes/dark.css'
          : '//cdn.jsdelivr.net/npm/docsify@4/lib/themes/vue.css';
        btn.textContent = isDark ? '☀️' : '🌙';
        localStorage.setItem(STORAGE_KEY, theme);
      }

      var saved = localStorage.getItem(STORAGE_KEY);
      if (saved === 'dark') setTheme('dark');

      btn.addEventListener('click', function() {
        var isDark = link.href.includes('dark');
        setTheme(isDark ? 'light' : 'dark');
      });
    })();
  </script>
</body>
</html>
```
