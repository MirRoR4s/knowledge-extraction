# CS 知识萃取

从计算机科学各子领域的英文技术文章、博客和文档中提炼核心知识点，输出高可读性的结构化中文摘要。

覆盖领域包括但不限于：操作系统、计算机网络、数据库、编程语言与编译器、分布式系统、安全与密码学、AI/ML、软件工程、DevOps、数据结构与算法等。

---

## 快速开始

向本项目输入一篇英文技术文章的 URL，即可自动生成结构化中文 Markdown 摘要文件。

### 分类导航

`articles/` 目录下按子领域组织，例如：

- [Vibe Coding](/articles/vibe-coding/)

> 侧边栏中「按标签浏览」区域支持跨分类按主题标签交叉检索所有文章。
